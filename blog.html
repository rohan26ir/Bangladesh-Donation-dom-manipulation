What is the Document Object Model (DOM)?

The Document Object Model is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree.

How do you select an element from the DOM?

The getElementById method allows you to select an element by its ID. This method returns the element object if it finds an element with the specified ID, otherwise, it returns null.

let element = document.getElementById("myId");
What is event delegation in the context of the DOM, and why is it useful?

Event delegation leverages event propagation to manage events efficiently. Instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. This listener takes advantage of event bubbling to handle events for its child elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes using the DOM, you can use setAttribute(), getAttribute(), or directly change the attribute (eg, element.id). For styles, the style attribute allows you to change the CSS (eg, element.style.color='blue'). You can also add or remove CSS classes using classList.add() or classList.remove().
